Web Basics

Front-end Development

Front-end development is the tier of full-stack development that deals with visual elements of an application that a user interacts with. The three languages that a browser can render are HTML, CSS, and JavaScript.
If you were to compare this tier of development to a a human body it would be the skin

HTML

HTML (Hypertext Markup Language) is used to define the structure ot the content of a webpage. It is sthe most important of the three wev languages because it provides content to the user and gives CSS and JavaScript somthing to targt. Without HTML there would be no page content.

CSS

CSS (Cascading Stylesheets ) is used for the styling, formatting, and layout of web pge content. It was introduced as a way of sepaating the cocerns of markup and styling into separate files. Prior to the introduction of CSS all styling was done in the individual HTML elements.

JavaScript

JavaScript allows us to make web pages more interactive and to add validation. Another important purpose is to give us the ability to perform programming logic in the front end and react to page events.

LoremIpsum

Lorem ipsum is sample text that is often used in the mockup phase of site developmnet as a means of laying content out on a page without having to have the final copy completed. It allows us to focus on how content will eventually lay out on the page rater than having to worry about the specific content. It is traditionally written in nonsense latin as a way to focus on the layout without getting distracted by the text. However, there are numerous lorem ipsum generators that will write fun nonsense content for you.
